# online-egitim-platformu
MERN Stack ve WebRTC mimarisiyle geliştirilen interaktif eğitim platformu

Bu proje öğrenciler ve öğretmenler için geliştirilmiş
etkileşimli bir online eğitim platformudur.

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- Canlı ders modülü
- Video konferans
- Ödev ve notlandırma sistemi
- EKLENECEK...

## Kullanılacak Teknolojiler
Frontend
- React

Backend
- Node.js
- Express

Database
- MongoDB
